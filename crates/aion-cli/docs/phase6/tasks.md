# Phase 6 — 打磨与发布 · 任务清单

## 目标

生产级体验，可作为默认客户端面向用户发布。

---

## 任务

### 6.1 响应式布局

- [ ] 窄终端（<80 列）自动隐藏 Team 侧面板
- [ ] 极窄终端（<40 列）降级为纯文本模式
- [ ] 动态 resize 无闪烁过渡

### 6.2 网络韧性

- [ ] 断线检测 + 状态栏 [disconnected] 警告
- [ ] 自动重连（指数退避）
- [ ] 重连后恢复会话状态

### 6.3 通知系统

- [ ] 终端 bell 通知（agent 回复完成）
- [ ] 配置项：启用/禁用 bell
- [ ] Team 成员状态变化通知

### 6.4 终端兼容

- [ ] 无 true-color 支持时退化为 256 色 / 16 色
- [ ] 无 Unicode 支持时退化为 ASCII 图标
- [ ] 主流终端测试：iTerm2、Terminal.app、Alacritty、Windows Terminal

### 6.5 配置系统

- [ ] 配置文件路径：~/.config/aion/config.toml
- [ ] 可配置项：默认 agent、默认 model、主题、bell、键位
- [ ] 配置热加载（可选）

### 6.6 自定义键位

- [ ] 键位映射配置格式设计
- [ ] 运行时键位重绑定
- [ ] 键位冲突检测与提示

### 6.7 稳定性

- [ ] panic hook：崩溃时恢复终端状态 + 打印错误
- [ ] 优雅退出：所有场景下恢复终端
- [ ] 大量消息虚拟滚动（性能）
- [ ] 内存使用监控（长会话不泄漏）
