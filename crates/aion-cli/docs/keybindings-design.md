# Aion CLI 快捷键设计

## 概述

所有操作可纯键盘完成，鼠标为辅助。快捷键按上下文分层：全局 > 模式 > 状态。

---

## 全局快捷键（任何模式下生效）

| 键位 | 操作 |
|------|------|
| `Ctrl+C` | 退出 Aion |
| `Ctrl+L` | 清屏 |
| `Tab` | 补全（指令/文件路径） |
| `Esc` | 上下文感知：关闭面板 > 中断回复 > 取消当前输入 |

---

## 输入区快捷键

| 键位 | 操作 |
|------|------|
| `Enter` | 发送消息 |
| `Shift+Enter` | 换行（多行输入） |
| `↑` / `↓` | 浏览历史输入 |
| `Ctrl+U` | 清空当前输入 |
| `/` | 触发指令面板（输入区为空时） |

---

## Home 模式

| 键位 | 操作 |
|------|------|
| `↑` / `↓` | 选择列表项 |
| `Enter` | 确认选择（恢复对话/进入团队） |
| `N` | 新对话 |
| `T` | 新团队 |
| `D` | 删除选中对话 |

---

## Chat 模式

| 键位 | 操作 |
|------|------|
| `Esc` | 中断 agent 当前回复 |
| `Page Up` / `Page Down` | 滚动消息区 |

---

## Team 模式

| 键位 | 操作 |
|------|------|
| `Esc` | 中断当前聚焦成员回复 |
| `Shift+↑` / `Shift+↓` | 切换聚焦成员（左侧面板） |
| `Shift+←` / `Shift+→` | 切换会话视图（全局 / 单成员） |
| `@` + `name` | 指定消息接收成员 |
| `Page Up` / `Page Down` | 滚动消息区 |

---

## 指令面板（`/` 触发后）

| 键位 | 操作 |
|------|------|
| `↑` / `↓` | 选择指令 |
| `Enter` / `Tab` | 确认选中指令 |
| `Esc` | 关闭指令面板 |
| 继续输入 | 模糊搜索过滤 |

---

## Esc 优先级规则

`Esc` 根据当前状态按以下优先级执行第一个匹配的操作：

1. 关闭指令面板（如果打开）
2. 关闭弹窗/浮层（如果存在）
3. 中断 agent 回复（如果正在流式输出）
4. 清空当前输入（如果有内容）
5. 无操作

---

## 设计原则

1. **最小记忆负担**：高频操作用直觉键位（Enter 发送、Esc 取消/中断）
2. **上下文感知**：同一个键在不同状态下做最合理的事
3. **不冲突**：避免和终端模拟器常用快捷键冲突（如 Ctrl+Tab、Ctrl+W）
4. **可配置**：未来支持用户自定义键位映射
