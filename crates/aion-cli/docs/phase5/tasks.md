# Phase 5 — Team 模式 · 任务清单

## 目标

多 agent 协作在 TUI 中可视化，可观察完整团队工作流。

---

## 任务

### 5.1 Team 布局

- [ ] 双栏布局：左侧成员面板 + 右侧聊天区
- [ ] 成员面板宽度固定（~15 字符）
- [ ] Team 状态栏：[Team: name] [mode] [N working / M idle] [session]

### 5.2 成员面板

- [ ] 成员列表渲染
- [ ] 状态图标实时更新：● Working / ○ Idle / ◉ 有新输出 / ✗ Error
- [ ] Shift+↑/↓ 切换聚焦成员（高亮）
- [ ] 成员数量超出面板时滚动

### 5.3 会话视图

- [ ] 全局视图：所有成员消息按时间交错，前缀发送者标识
- [ ] 单成员视图：Shift+←/→ 切换，只显示该成员的对话
- [ ] 视图切换时保持滚动位置

### 5.4 多 Agent 流式

- [ ] 并行接收多个成员的流式输出
- [ ] 按到达顺序追加渲染
- [ ] Esc 中断当前聚焦成员的回复

### 5.5 消息路由

- [ ] @mention 语法解析
- [ ] 指定成员发送消息
- [ ] 无 @ 时发送到全局（Leader 处理）

### 5.6 Team 生命周期

- [ ] `aion team chat <ID>` 进入已有团队
- [ ] 团队创建流程（通过指令或 Home）
- [ ] 成员加入/离开事件处理
