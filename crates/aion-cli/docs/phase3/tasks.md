# Phase 3 — 完整 Chat 模式 · 任务清单

## 目标

Chat 模式达到日常可用，体验接近设计稿。

---

## 任务

### 3.1 布局系统

- [ ] 三段式布局：状态栏（顶部 1 行）/ 消息区（中间弹性）/ 输入区（底部）
- [ ] 状态栏渲染：[agent · model] + [session: short_id]
- [ ] 终端 resize 事件处理 + 重新布局

### 3.2 消息渲染增强

- [ ] Markdown 内联渲染：加粗、斜体、行内代码
- [ ] 代码块语法高亮（集成 syntect 或 tree-sitter-highlight）
- [ ] 代码块语言标签显示
- [ ] 工具调用折叠区块渲染（工具名 + 参数摘要）
- [ ] 错误消息红色标记
- [ ] 发送者标识样式（You: / AgentName: / [system]）

### 3.3 输入区增强

- [ ] 多行输入模式（Shift+Enter 触发换行）
- [ ] 输入历史：↑/↓ 浏览已发送消息
- [ ] Ctrl+U 清空当前输入
- [ ] 输入区高度自适应（多行时扩展）

### 3.4 滚动与导航

- [ ] Page Up / Page Down 滚动消息区
- [ ] 新消息到达时自动滚到底部（除非用户正在查看历史）
- [ ] 滚动位置指示

### 3.5 流式交互

- [ ] Esc 中断 agent 当前回复
- [ ] 流式输出期间输入区保持独立可编辑
- [ ] 流式光标 ▌ 渲染
